Como abrir o projecto:
(com script)

 - Abrir o PADI-FS
 - Colocar os scripts na pasta scripts
 - Abrir o PADIFS.cs
 - Fazer clean e build da solução
 - Fazer f5 (correr)
 - Carregar em lauch script
 - Escolher o script
 - Seleccionar o script na list box
 - Carregar em Run All ou Run Step
 - Para fechar: Kill All
